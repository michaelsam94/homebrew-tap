# homebrew-tap

Homebrew tap for [mac-cleaner](https://github.com/michaelsam94/mac_cleaner).

## Install

```bash
brew tap michaelsam94/tap
brew install mac-cleaner
```

## Upgrade

```bash
brew update
brew upgrade mac-cleaner
```

## Uninstall

```bash
brew uninstall mac-cleaner
brew untap michaelsam94/tap
```
